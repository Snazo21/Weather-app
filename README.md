# Weather-app
Basic app for finding weather by searching for your city/town
## Features
* **Current Weather:** Real-time display of temperature, conditions, and atmospheric data.
* **Forecast Table:** A clean breakdown of upcoming weather trends for 5 days, starting with current day.
* **Dynamic Backgrounds:** UI changes based on weather conditions (Cold, Warm, Rainy).
* **Error Handling:** Robust data validation to ensure the app doesn't crash during API downtime.

## Tech Stack
* **Framework:** - Vue.js / Vite
* **API:** - OpenWeatherMap API
* **Styling:** - CSS
* **Icons:** - FontAwesome
